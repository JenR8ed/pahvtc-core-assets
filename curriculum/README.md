# Master Curriculum

This folder contains the **26-Week Master Curriculum** for PAHVTC vocational training programs.

## Structure
- **SOPs/** – Standard Operating Procedures for each phase
- **Phase-1/** – Weeks 1–7: Foundation & Orientation
- **Phase-2/** – Weeks 8–13: Core Skills Development
- **Phase-3/** – Weeks 14–20: Applied Practice & Specialization
- **Phase-4/** – Weeks 21–26: Capstone, Certification & Placement

## Overview
The curriculum is designed as a competency-based, culturally affirming vocational pathway aligned with state workforce development standards.
